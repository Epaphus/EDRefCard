Elite: Dangerous Reference Card Generator

Create and/or share a graphical reference card for your keyboard and controller bindings.

Once you have set up your bindings in the game you will find them as a 'Custom.2.0.binds' file in your local data directory (typically %LOCALAPPDATA%\Frontier Developments\Elite Dangerous\Options\Bindings).

You can then input this file below to create and/or share a reference card.
