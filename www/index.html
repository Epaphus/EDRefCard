EDRefCard

Create and/or share a graphical reference card for your keyboard and controller bindings for Elite: Dangerous.

View a list of published bindings. Searching by criteria such as date range, controller type etc is coming soon.

You can find your bindings file as a '.binds' file in your local data directory (typically %LOCALAPPDATA%\Frontier Developments\Elite Dangerous\Options\Bindings).

You can then upload this file below to create and/or share a reference card.
